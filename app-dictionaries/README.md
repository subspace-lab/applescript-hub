# App Dictionaries

Pre-extracted and human-readable documentation of scriptable app dictionaries.

## Generating a Dictionary

```bash
# Raw SDEF (XML)
sdef /Applications/AppName.app > app-dictionaries/appname.sdef

# For browsing in Script Editor
# File → Open Dictionary → select .sdef file
```

## How Docs Are Generated

The markdown files here are auto-generated by [`tools/sdef_to_md.py`](../tools/sdef_to_md.py).
The script reads the SDEF directly from the app bundle — raw SDEF files are never committed.

```bash
# Generate docs for any app
uv run tools/sdef_to_md.py "Finder" --out app-dictionaries/finder.md
uv run tools/sdef_to_md.py "Mail"   --out app-dictionaries/mail.md

# Or from a .sdef path directly
uv run tools/sdef_to_md.py --path /path/to/app.sdef --out output.md
```
